### This is a project I started in 2025 to help me learn C++

In the beginning I wanted to create a working n-body simulation within a reasonable error range, where you could add planets on a map, then set their starting velocities and observe some 2D orbital mechanics.

---

The concept already works and the simulation is numerically stable with a relative system energy change of ~0.25%.  
This of course is only possible with good starting conditions (no near passes of planets, etc.).

I am using basic verlet integration for simulating the orbits.  
Hence the error at simulation start is a bit higher and tends to go down.

---

As the project went on and I learned more and more concepts of C++,  
I also watched how the project got bigger and bigger and more and more features needed to be added to achieve my vision.

Right now the project is nowhere close to being done.  
And knowing myself it probably won't ever be, because I will always find things I would love to add to perfect it.

---

Nevertheless I am absolutely happy that I spent many hours of my time learning C++.

If you try out the code (and manage to run it) feel free to give me some feedback — I'd be very glad!
